# Proyecto: Ruzafa App

## Objetivo
Aplicación web para gestionar finanzas de un emprendimiento de collares.

Debe permitir:
- ventas, costos, gastos, retiros
- resumen mensual
- configuración financiera editable
- historial de reglas
- asistente interno basado en datos

## Stack
- Next.js
- TypeScript
- Tailwind
- Supabase
- Netlify

## Reglas clave
1. No hardcodear valores financieros
2. Configuración siempre editable
3. Historial de configuraciones
4. Cambios no afectan meses pasados
5. Mobile-first
6. No usar IA externa aún

## Fórmulas
- utilidad = ventas - costos - gastos
- disponible = utilidad - retiros
